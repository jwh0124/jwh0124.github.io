## About Mono ##

> Mono는 .Net 프레임워크(framework)의 오픈 소스 개발 플랫폼으로서, 크로스-플랫폼(cross-platform) 어플리케이션의 개발을 지원한다. C#과 CLI(Common Language Infrastructure)에 기반을 두고 있다.

### Mono install ###

#### Raspberry Pi 4 ####

```bash
sudo apt-get install mono-complete
```
