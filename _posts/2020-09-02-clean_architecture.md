## Clean Architecture ##

> Clean Architecture Review

### Clean Architecture Definition ###

- 모든 소프트웨어 시스템은 행위와 구조 두 가지 가치를 제공한다.

### Paradigm List ###

- 구조적 프로그래밍 : 제어흐름의 직접적인 전환에 대해 규칙을 부과한다.
- 객체 지향 프로그래밍 : 제어 프름의 간접적인 전환에 대해 규칙을 부과한다.
- 함수형 프로그래밍 : 할당문에 대해 규칙을 부과한다.
