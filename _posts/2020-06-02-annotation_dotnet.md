## Used Dotnet Annotation ##

### EntityFramework Core ###

> EntityFramework Core 개발 시 사용한 Annotation

| Annotation | Description |
| :--- | :--------------------------------------------------------- |
| `@DatabaseGenerated` | 데이터베이스에서 속성 값을 생성하는 방법을 지정합니다. (None, Identity, Computed) |
| `@Table` | 테이블 선언 Annotation |
