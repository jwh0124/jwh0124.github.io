### Golang Package ###

| Package     | Description |
| :---------- | :---------- |
| `net/http`  | HTTP 클라이언트와 서버를 작성하는데 필요한 기능을 제공 |
| `encoding/json` | JSON과 Go 타입 사이의 인코딩 및 디코딩 작업을 빠르고 쉽게 수행해주는 Package |
