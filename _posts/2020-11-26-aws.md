## AWS ##

> Amazon Web Service

### Install AWS cli ###

> AWS Conmmand Line Interface

```bash
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```

### AWS Access Infomation Configure Command Line ###

```bash
# Command 창을 통해 접속 -> Access Key, Secret Key, Region 필요
aws configure
```
