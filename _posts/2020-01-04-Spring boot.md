## Spring Boot ##

### Spring Boot 란 ? ###
> 실행만 하면 스프링 기반의 상용화가 가능한 애플리케이션을 쉽게 만들 수 있도록 만든 스프링 프레임워크.


### Spring Boot 특징 ###
- 단독 실행(Stand-alone) 가능
- 내장된 톰캣, 제티 등의 서버를 이용하여 별도의 서버 설치 없이 실행 가능
- 자동화 된 설정 방식 제공
